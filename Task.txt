
В городе Токаномир есть несколько больниц для животных.
В связи со вспышкой бешенства больницы перестают справляться с количеством поступающей информации.
За этот месяц поступило уже 9820 животных, которым нужна помощь!

Как можно скорее помогите больницам Токаномира и разработайте программу для учёта больных животных!

0. Данная программа должна уметь получать текстовую базу, в которой будут в каждой строке описаны по прорядку: кличка, класс, заболевание, наличие прививки, дата поступления в больницу.

Пример:
	Барбос;собака;бешенство;Нет;20.10.2019
	Макс;собака;бешенство;Да;1.11.2019
	Барсик;кошка;укусы;Нет;29.5.2019
	Вольт;попугай;бешенство;Да;21.3.2020

1. Программа должна уметь принимать запрос для поиска животных и выводить все соответствующие результаты:

	- по классу животного (по запросу собака вывести всех собак с описанием их характеристик и их количество),
	- по дате поступления (при вводе даты вывести всех животных, поступивших на эту дату и их количество)
	- по болезни (при вводе диагноза вывести животных с данным диагнозом и их количетство)
	- по наличию прививки (при запросе вывести всех животных имеющих или не имеющих прививки и их количество)
	- по кличке (поиск животного по кличке и вывод всех его характеристик и их количество)

Примеры запроса:
	Класс - собака
	Кличка - Барсик
	Дата - 20.10.2019

БОНУСЫ (есть смысл выполнять только, если основная часть программы работает идеально):
2. Программа должна определять, является ли дата поступления действительной, и в противном случае перезаписывать базу без животных с недействительной датой.

3. Программа должна выводить животных, у которых есть прививка зелёным цветом, а тех, у кого её нет, красным цветом

4. Программа должна иметь возможность включать механику, описанную в третьем пункте.
Например, механика будет работать, только если запустить программу с параметром -color:
	$> python myprogram.py -color
